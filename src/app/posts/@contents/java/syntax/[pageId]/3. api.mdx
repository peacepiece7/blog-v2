# maven, gradle

아직 잘 모르겠지만 mvnrepository.com 에서 (이하 mvnrepo) 라이브러리를 검색하고 사용하면 된다.

npm 같은 java pakcage manager 같다.

npm install시 node_modules에 패키지가 추가되는 것 처럼 .jar로 끝나는 패키지를 받아서 직접 넣는 방법이 있고

script 태그를 추가하듯이 xml 태그를 복사해서 pom.xml에 추가하는 maven 이라는 방식이 있고

모바일 앱 개발시에는 gradle에 추가하는 방식이 있다.

maven은

pom.xml에 이런 내용을

```xml
<!-- https://mvnrepository.com/artifact/com.google.code.gson/gson -->
<dependency>
    <groupId>com.google.code.gson</groupId>
    <artifactId>gson</artifactId>
    <version>2.11.0</version>
</dependency>
```

gradle은

build.gradle에 이런 내용을 넣어주며 된다.

```
// https://mvnrepository.com/artifact/com.google.code.gson/gson
implementation group: 'com.google.code.gson', name: 'gson', version: '2.11.0'
```
