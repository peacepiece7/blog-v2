# installation

pnp를 지원하지 않기 떄문에 pnpm을 선택

```bash
corepack enable
npm i -g pnpm
pnpm dlx create-turbo@latest
```

설치하는데 생각보다 오래걸림 멈춘건줄 알고 어러번 캔슬했는데, 웹 서핑좀 하고 오니까 설치 진행중이라고 나옴..

# Telemetry

익명으로 사용자 데이터를 수집한다고 함. 비활성화 가능

제품에서 쓸 계획이라면 비활성화 할 것 같은데 일단 알아만 두자.

`turbo telemetry status`
